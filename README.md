## How run local?

you will need [.NET](https://dotnet.microsoft.com/en-us/download) installed:

On windows PowerShell execute:

```powershell
$ ./docs/build.ps1
```

Alternatively

```sh
$ dotnet tool install -g docfx
$ cd ./docs
$ docfx --serve

```
